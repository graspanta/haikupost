==========
Haikupost
==========

概要
-----

Haikupostは、俳句の投稿サービスです。

機能一覧
-------

* ユーザ登録
* ユーザログイン
* 一覧表示
* 詳細表示
* 俳句投稿
* ユーザ名と単語検索
* ページネーション
* 画像アップロード
